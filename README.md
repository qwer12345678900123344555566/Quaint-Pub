# 小酒馆

一个本地运行的 AI 聊天页面，用于测试与使用多种兼容 OpenAI 接口的模型。支持主题、头像、背景、系统提示词（角色设定）等扩展功能。

## 🌐 在线体验
**👉 点此体验：  
[AI聊天](https://sgame-data-service-1252931805.cos.ap-nanjing.myqcloud.com/SnsShare/YMZ/OpenAITest.html)**

---

## 🚀 使用方法

### 1. 启动
直接打开文件 **OpenAITest1.html**。

---

## 配置模型（必要）

进入右上角 **设置 → AI 提供商配置**，按以下填写：

- **提供商名称**：自定义，如 `OpenAI`
- **Base URL**：如  
  - `https://api.openai.com`  
  - `https://openrouter.ai/api`  
  - `http://127.0.0.1:1234`
- **API 路径**：一般为 `/v1/chat/completions`
- **API Key**：你的密钥
- **模型名称**：输入模型 ID，示例：  
  - `gpt-4.1-mini`  
  - `deepseek-chat`  
  输入后点右侧 **＋** 添加。

也可使用 **“获取模型”** 自动从 `/v1/models` 读取模型列表。

---

## 选择模型

在页面顶部的 **AI 模型** 下拉框中选择要使用的模型。

---

## 系统提示词（角色扮演）

在 **设置 → 系统提示词** 中可以：

- 添加酒馆风格的人物设定  
  示例用途：  
  - “酒馆老板”  
  - “旅人接待员”  
  - “吟游诗人讲述者”  
- 设置默认在新对话中启用   
- 可使用外部模型（如 Grok）生类似酒馆的提示词粘贴进来（grok可生成nfsw的提示词）

这些提示词会作为系统消息注入对话，实现更强的角色体验。

---

## 更换聊天背景

在 **设置 → 聊天背景** 上传任意图片即可。  
背景仅存储在本地浏览器。

---

## 自定义助手头像与昵称

在 **设置 → 助手信息** 中可以：

- 上传头像  
- 修改昵称  

所有内容本地保存。

---

## 其他功能

- **聊天历史**：自动保存，可在右上角“历史”查看  
- **附件上传**：支持图片、文本文件  
- **主题切换**：暗色 / 亮色 / 蓝 / 绿 / 紫 / 橙  
- 全部数据保存在浏览器 `localStorage`，无需服务器
